# 05 — První continuous movement měření

Datum/čas: 2026-06-04 23:01.

## Záznam

LiDAR byl zapnutý, držený v ruce a čtený přes USB/serial `/dev/ttyACM0`.

Příkaz:

```bash
cd /path/to/unitree-lidar
FRAMES=80 DURATION_SECONDS=15 PACKETS_PER_FRAME=20 \
  measurements/continuous_movement/record_sequence.sh --port /dev/ttyACM0
```

Výstupní sekvence:

```text
scans/seq_20260604_230117/
scans/latest_sequence -> seq_20260604_230117
```

Naměřeno:

```text
framy: 80
point packety: 1600
IMU packety: 1857
ACK: 1
body zapsané ve framech: 473642
velikost sekvence: cca 7.5 MB
```

## Rekonstrukce mapy

Plný KISS-ICP přes `build_map.sh` zatím nešel spustit, protože `.venv` nemělo nainstalované `open3d`/`kiss_icp` a pip neměl DNS přístup k internetu.

Proto byl použit fallback:

```bash
measurements/continuous_movement/build_map_simple.sh
```

Výstup:

```text
results/continuous_map_simple_20260604_230412.pcd
results/continuous_map_simple_20260604_230412.png
results/continuous_map_simple_20260604_230412.report.csv
results/latest_map.pcd -> continuous_map_simple_20260604_230412.pcd
results/latest_map.png -> continuous_map_simple_20260604_230412.png
```

Výsledek fallback rekonstrukce:

```text
body po voxel downsample: 168171
rozměr X x Y x Z: 7.20 x 6.77 x 4.25 m
PCD: cca 2.7 MB
PNG: cca 0.9 MB
```

## Poznámka ke kvalitě

Fallback používá jednoduché 2D ICP přes `numpy/scipy`, ne plnohodnotný KISS-ICP.
Je to rychlý pracovní náhled, ne finální SLAM. Pro lepší výsledek později doinstalovat `.venv` a spustit KISS-ICP cestu.
